The Level Up App

A innovative ASP.Net Core web application developed collaboratively over three weeks with my teammates Eli Paris and Rafi Wick, divided into three distinct sprints. This platform enhances productivity through sophisticated task management and engaging gamification features. Users can maintain organized lists of tasks, customize tasks aligned with their personal aspirations, and experience gamified progression by "leveling up" their account upon task completion.

Features

AI-Generated Profile Pictures: One of the standout features of Level Up is the AI-generated profile pictures. Using OpenAI's powerful API, these images evolve to visually represent user progress over time. This unique approach not only personalizes the experience but also serves as a motivational tool, encouraging users to continue engaging with the app and tracking their development. Dynamic Radar Graph: To further enhance user interaction, Level Up includes a dynamic radar graph that provides a visual summary of weekly achievements. This feature allows users to quickly assess their progress and areas for improvement, promoting regular interaction and sustained engagement with the platform.

BackEnd

Backend Technology: The backbone of Level Up is built using C#.Net and ASP.Net Core, technologies chosen for their reliability and scalability. This robust framework supports the intricate features of the application while ensuring a smooth user experience. Database Management: PostgreSQL was selected for database management due to its powerful capabilities and scalability. It effectively handles the complex data needs of Level Up, ensuring data integrity and fast query responses, which are crucial for maintaining an excellent user experience.
